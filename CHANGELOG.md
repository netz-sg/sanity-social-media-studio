# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-01-01

### Added
- Initial release
- Graphics Studio with 5 built-in styles (Industrial, Minimal, Gradient, Bold, Neon)
- Multi-platform social media posting (Instagram, Facebook, Threads, X)
- Post scheduling with timezone support
- Monthly post calendar view
- CSV import for bulk post creation
- Batch image generation
- Draft and template management system
- Document actions for concerts, news posts, and aftershow stories
- Concert Day Graphics Generator
- Sanity schemas for templates and Late API settings
